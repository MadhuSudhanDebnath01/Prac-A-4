Display: how a particular element need to be display

1.Inline 2.Block 3.Inline block 4.None 5.Hidden 6.flex

Inline : it is a property in display which will occupy how much the content of the element is required.it will come in same line , we can't apply height & width if we apply also no use.

Block :it a property in display which will occupy the complete width and it will come in next line we can apply height and width.

Inline-Block : it is a property in display its a combination of both inline & block , we can apply height & width and margin and padding.

None : which is removed completely from our document or our browser screen.

Flex : Flex is an one directional display property. Either we can have row or coloumn only and we can manipulate our items.

Justify contents : It will act on x-axis or y-axis.

Align items : Act on x-axis or y-axis.
